# 公众号运营体系 Skills

基于 slavingia/skills（Gumroad 创始人极简创业方法论）的设计范式，为公众号运营打造的 13 个 Agent Skills。

## 四大模块

### 🔍 前期准备（3个）

| Skill | 用途 |
|-------|------|
| **选题发现** | 信息差选题，评估框架，标题公式 |
| **语料学习** | 30话题画廊，历史文章风格学习，Playbook自动生成 |
| **信息采集** | 全网素材收集，多源交叉验证 |

### ✍️ 内容生产（3个）

| Skill | 用途 |
|-------|------|
| **内容写作** | 洋葱结构，完读率优先，Playbook驱动 |
| **SEO/GEO** | 搜一搜优化 + AI搜索引擎优化 |
| **配图生成** | 截图优先，AI封面，豆包API |

### 🚀 发布优化（5个）

| Skill | 用途 |
|-------|------|
| **排版主题** | 8主题画廊，点选切换，设为默认 |
| **排版发布** | WenYan排版 → 合规 → 推草稿 → 冷启动 |
| **合规审核** | 16项自动检查（红线+必须+优化） |
| **冷启动** | 前2h生死期操作，流量池闯关 |
| **跨平台** | 公众号 → 知乎/小红书/视频号分发 |

### 📊 复盘迭代（2个）

| Skill | 用途 |
|-------|------|
| **数据复盘** | 48h考核期分析，优化建议 |
| **增长复盘** | 全局运营决策校验 |

## 运营路径

```
选题发现 → 语料学习 → 信息采集 → 内容写作 → 配图生成 → 排版主题 → 排版发布 → 合规审核
    ↓              ↓                                                                    ↓
增长复盘 ← 数据复盘 ← 跨平台分发 ← 冷启动 ← 发布后2h
```
